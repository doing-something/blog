행렬이 점/좌표를 어떻게 이동시키는가를 2x2 사례로 시각화하면서 — 벡터, 내적, 행렬-벡터 곱, 기저벡터, determinant, 기본 선형변환, 좌표계 변환까지 — 선형대수의 기초 개념을 배운다.

- [코드](https://github.com/doing-something/playground/tree/main/matrix-transform)

## 내적 (dot product)

두 벡터가 얼마나 같은 방향(같은 성격)인가를 하나의 숫자로 측정하는 연산.
같은 축에서 두 벡터의 성분 부호가 같으면 곱이 양수, 반대면 음수가 된다. 모든 축의 곱을 더한 값이 두 벡터의 방향 일치도다.

```
θ = 0°    → cos(0°)  = 1    // 내적 최대 (양수)
θ = 90°   → cos(90°) = 0    // 내적 0
θ = 180°  → cos(180°)= -1   // 내적 최소 (음수)
```

방향 일치도는 cos(θ), 그 방향에 실린 양은 |A|×|B|. 둘을 곱한 값이 내적이다. 크기를 1로 정규화한 벡터의 내적은 cos(θ) 그 자체가 되며, 이를 코사인 유사도라고 한다.

```
A · B = a1×b1 + a2×b2 + ...    // 좌표 정의
A · B = |A| × |B| × cos(θ)     // 기하학적 정의
```

## 행렬은 입력 벡터를 출력 벡터로 바꾸는 식을 숫자로 적은 것

`2x2` 행렬과

```text
[[a, b],
 [c, d]]
```

벡터를

```text
[x, y]
```

곱하면 새 좌표는

```text
새 x = a*x + b*y
새 y = c*x + d*y
```

가 된다.

행렬이 하는 일은 "곱하고 더해서 새 좌표 만들기"다.

## 행렬은 왜 값을 바꾸는 도구로 쓰이나?

확대, 회전, 기울이기를 식으로 쓰면 같은 모양이다.

```
2배 확대:
새 x = 2×x + 0×y
새 y = 0×x + 2×y

30도 회전:
새 x = 0.87×x + (-0.5)×y
새 y = 0.5×x  +  0.87×y

x방향 기울이기:
새 x = 1×x + 1×y
새 y = 0×x + 1×y
```

전부 같은 패턴이다.

```
새 x = a×x + b×y
새 y = c×x + d×y
```

원래 좌표를 곱하고 더하는 것 외에는 하지 않는다. 변환을 식별하는 정보는 a, b, c, d 네 개뿐이고, 이 네 개를 담은 것이 행렬이다.

```
[[a, b],
 [c, d]]
```

곱하고 더하기로 표현 가능한 변환을 선형 변환이라고 한다. 종류가 무엇이든 "행렬 × 벡터" 하나의 형태로 적용된다.

### 행렬 × 벡터 = 행과 벡터의 내적

```
[[a, b],   [x]     [a×x + b×y]
 [c, d]] × [y]  =  [c×x + d×y]
```

행렬 x 벡터 계산은 "각 행과 벡터를 한 번씩 내적한다."

## 행렬 읽는 법 — 각 열은 기저벡터의 새 좌표

```
A = [[2, 0],
     [0, 2]]

1열 [2, 0] // x축 단위벡터 (1, 0)이 (2, 0)으로 간다  → x축이 2배
2열 [0, 2] // y축 단위벡터 (0, 1)이 (0, 2)로 간다  → y축이 2배
```

(1, 0)에 행렬을 곱하면

```
[a×1 + b×0,  c×1 + d×0]  =  [a, c]   ← 1열
```

(0, 1)에 곱하면 [b, d] — 2열.

행렬의 각 열은 그 자리에 있던 기저벡터가 어디로 옮겨지는지를 담고 있다.

```
회전:
[[0, -1],   // 1열 [0, 1]: x축이 위로 (90도 회전)
[1,  0]]    // 2열 [-1, 0]: y축이 왼쪽으로

기울이기:
[[1, 1],    // 1열 [1, 0]: x축 그대로
[0, 1]]     // 2열 [1, 1]: y축이 오른쪽으로

반사:
[[-1, 0],   // 1열 [-1, 0]: x축이 좌우 반전
[ 0, 1]]    // 2열 [ 0, 1]: y축 그대로
```

## determinant는 넓이 변화와 정보 손실을 보여준다

```
det([[a, b],
     [c, d]]) = a×d - b×c
```

- |det| = 면적이 변하는 비율
- 부호 = 방향이 뒤집혔는가 (음수면 반사 성분 있음)
- det = 0 = 한 직선으로 눌려 면적이 0, 역행렬이 존재하지 않는다.

```
[[2, 0], [0, 2]]    det = 4    면적 4배
[[0,-1], [1, 0]]    det = 1    회전 — 면적 보존
[[1, 1], [0, 1]]    det = 1    기울이기 — 면적 보존
[[-1,0], [0, 1]]    det = -1   좌우 뒤집힘
[[1, 2], [2, 4]]    det = 0    한 직선으로 눌림
```
